    cat指令

    cat命令用于连接文件并打印到输出设备上

    语法

    选项

    -n 或 –number：由 1 开始对所有输出的行数编号。

    -b 或 –number-nonblank：和 -n 相似，只不过对于空白行不编号。

    -s 或 –squeeze-blank：当遇到有连续两行以上的空白行，就代换为一行的空白行。

    -v 或 –show-nonprinting：使用 ^ 和 M- 符号，除了 LFD 和 TAB 之外。

    -E 或 –show-ends : 在每行结束处显示 $。

    -T 或 –show-tabs: 将 TAB 字符显示为 ^I。

    head指令

    head指令可用于查看文件开头部分的内容，有一个常用的参数 -n用于显示行数，默认显示10行

    语法

    head [选项] [文件]

    参数

    -q 隐藏文件名

    -v 显示文件名

    -c<数目> 显示的字节数。

    [注] 当数目数为负数n时，表示输出除了最后n个字符外的全部内容

    -n<行数> 显示的行数。

    [注] 当行数为负数n时，表示输出除了最后n个行外的全部内容

    demo

    输出temp文件头10行至input文件

    head temp > input

    输出temp文件前5行

    head -n 5 temp

    输出文件前20个字节

    head -c 20 temp

    参考

    https://www.runoob.com/linux/linux-comm-head.html

    tail指令

    tail指令可用于查看文件的内容，将文件的最尾部内容显示在屏幕上

    语法

    参数

    -f 循环读取

    –pid=PID 与-f合用,表示在进程ID,PID死掉之后结束

    -s, –sleep-interval=S 与-f合用,表示在每次反复的间隔休眠S秒

    -q 不显示处理信息

    -v 显示详细的处理信息

    -c<数目> 显示的字节数

    -n<行数> 显示文件的尾部 n 行内容

    +n 从第n行开始，显示文章的全部内容

    touch指令

    touch命令用于修改文件或者目录的时间属性，包括存取和更改时间，若文件不存在，系统会建立一个新的文件。

    语法

    选项

    -a：改变档案的读取时间记录

    -m：改变文档的修改时间记录

    -c：加入目的文档不存在，则不会建立新的文档

    – no-create：加入目的文档不存在，则不会建立新的文档

    demo

    修改文件testfile的时间属性为当前系统时间：

    touch testfile

    在当前目录下创建file1、file2、file3、file4、file5：（假设文件不存在）

    touch file{1..5}

    patch指令

    Linux patch命令用于修补文件。

    语法

    选项

    -b/–backup：备份每一个原始文件

    -B<备份文件名前缀>/–prefix<备份文件名前缀>：设置备份文件文件名的前缀

    -o<文件名>/–output<文件名>：修补后的文件名

    demo

    将文件file1打补丁，patch为升级后的文件：

    先生成补丁文件：

    diff file1 patch > file.patch

    利用补丁文件将file1补丁：

    patch -b file1 file.patch

    重定向符

    符号 作用
    命令 < 文件 将文件作为命令的标准输入
    命令 << 分界符 从标准输入中读入，直到遇到分界符为止
    命令 > 文件 将标准输出重定向到文件中（清除原有文件中的数据）
    命令 2> 文件 将错误输出重定向到文件中（清除原有文件中的数据）
    命令 >> 文件 将标准输出重定向到文件中（在原有的内容后追加）作用
    命令 2>> 文件 将错误输出重定向到文件中（在原有内容后面追加）
    参考

    https://www.cnblogs.com/walkwaters/p/12459727.html

    chmod命令

    该命令用来给文件增加调用权限限制

    语法

    chmod [+-=] [rwx] 文件名

    +：表示增加权限

    -：表示取消权限

    =：表示唯一设定权限

    r：表示可读取

    w：表示可写入

    x：表示可执行

    参考

    https://www.runoob.com/linux/linux-comm-chmod.html
