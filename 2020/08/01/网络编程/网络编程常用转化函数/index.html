常用网络字节转化函数

John Doe
2020-08-01 (Updated: 2021-12-12)
网络编程
项目笔记

u_short htons(u_short hostshort), u_long htonl(u_long hostlong);

​ 主机序转换至网络序

​ h: host, n: network, s: unsigned short, l: unsigned long

u_short ntohs(u_short hostshort)), u_long ntohl(u_long hostlong);

​ 网络序转换为主机序

u_long inet_addr(const char *cp);

​ 将点分十进制ipv4地址转换为整数,，转换失败返回INADDR_NONE

char *inet_ntoa(in_addr in);

​ 将addr_in内保存的ip转换为点分十进制字符串，返回的指针为静态指针，每次调用都会覆盖上次的返回结果

int inet_pton(int family, const char *pszAddrString, void *dst)；

​ 将点分十进制ip地址转化为网络字节整数

  • family: 协议族

    • AF_INET: ipv4
    • AF_INET6: ipv6

  • pszAddrString: 点分十进制ip地址字符串

  • dst: 存放转换结果

  • 转换成功返回1，转换失败返回0

const char* inet_ntop(int family, const void *addrprt, char *strptr, size_t len)；

​ 将网络字节整数转化为点分十进制ip地址

  • family: 协议族

    • AF_INET: ipv4
    • AF_INET6: ipv6

  • addrprt: 网络字节整数

  • strptr: 存放ip地址缓冲区

  • len: 缓冲区大小

  • 转换成功返回strptr，转换失败返回空指针
