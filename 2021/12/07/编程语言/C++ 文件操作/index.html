std::fstream
构造函数
类型 声明 说明
default (1) fstream(); 默认构造，不关联任何文件
initialization (2) explicit fstream (const char* filename, ios_base::openmode mode = ios_base::in | ios_base::out);
explicit fstream (const string& filename, ios_base::openmode mode = ios_base::in | ios_base::out);		 指定文件路径与打开方式
mode 含义 作用
in input 只读
out output 只写，覆盖方式，不存在则创建（前提是父目录都存在）
binary binary 二进制方式操作
ate at end 初始位置：文件（app包含此打开方式）
app append 所有输出附加在文件末尾
trunc truncate 如果文件已存在则先删除该文件
