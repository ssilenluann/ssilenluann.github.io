Python 常用魔术方法

John Doe
2023-07-14 (Updated: 2023-07-14)
编程语言
Python

简介

​ Python魔术方法(special method)是python提供的对类进行客制化定制的方法，是定义在类中的一些特殊方法，其命名特点就是方法名前后各有两个下划线__， 因此也被成为Dunder method (double underline method)。

基础相关

_new_(cls, x)

​ 从类创建对象时调用，该函数是有返回值的，必须返回类对象

_init_(self, x)

​ 初始化类对象时调用，无返回值

_del_(self)

​ 可以简单理解为析构函数，但是其触发时间不固定，有可能在对象引用为0的时候被调用，有可能在垃圾回收时被调用，也有可能在解释器结束的时候调用，注意，调用del关键字未必会触发__del__方法，因为del只是会减少对象的引用

_repr__(self), __str_(self)

​ 这两个函数功能是相似的，都是返回obj的字符串表示，区别主要在于，一般来说__str__主要返回可读性更强的string, __repr__主要返回更详细的信息，两个方法都定义了的情况下，print函数优先使用__str__方法

_format_(self, spec)

​ 这个方法主要用于print时的格式化输出

比较相关

method description
_eq_(self, other) ==
_ne_(self, other) !=
_gt_(self, other) >
_lt_(self, other) <
_ge_(self, other) >=
_le_(self, other) <=
_hash_(self) 必须返回整数值作为类对象hash值，相等对象的hash值必须相等，重写__eq__方法之后，默认__hash__方法会被删除，需要的话需要重新实现
_bool_(self) bool运算符

匹配规则

  • 子类运算符优先级高于父类

  • 运算符左侧高于右侧，除非右侧是子类

  • <=需要额外实现，’<=’ != ‘< or ==’

属性相关

_getattr_(self, name), __getattribute__(self, name)

​ 当获取对象的属性不存在时调用_getattr_, 只要尝试获取对象属性就会调用__getattribute__， 同时，实现__getattribute__时需要避免无限递归

_setattr_(self, name, val)

​ 当对类对象成员进行赋值操作时，会调用此方法

_delattr_(self, name)

​ 当尝试删除类对象属性的时候调用该方法

_dir_(self)

​ 当调用dir()方法时，此方法被调用，默认返回所有属性名，__dir__必须返回sequence

_slots_

​ 类级别的成员属性，类似白名单机制，存放允许类对象自定义的属性

类构建相关

_init_subclass_(cls)

​ 定义子类时，会调用父类的_init_subclass_()方法

_set_name_(self, owner, name)

​ 当某个类的对象作为其他类属性的值时，该类中的__set_name__会被调用

_class_getitem_(cls, item)

​ 当对类对象以下标的形式取值时，会调用此方法
